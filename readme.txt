== Installation ==

**Automatic Installation**

1. From your Dashboard go to Plugins > Add New
1. Search for "Loginstyle"
1. Click "Install Now"

**Manual Installation**

1. Download the plugin .zip file
1. On your Dashboard go to Plugins > Add New
1. Select "Upload Plugin" from the top of the page
1. Click "Choose File" and select the `loginstyle.x.x.x.zip` file from your computer and click "Install Now"
1. After the plugin has been successfully installed you can press "Activate Plugin" to start using the plugin

**FTP Installation**

1. Download the plugin .zip file and extract it to a location on your computer
1. Login to your FTP server
1. Upload the `loginstyle` folder to your `/wp-content/plugins/` directory
1. Activate Loginstyle under Dashboard > Plugins > Installed Plugins

== Frequently Asked Questions ==

= How can I get support? =

For bug reports please leave a post on the Support page for this plugin. For dedicated support please consider upgrading to the [Premium or Developer](http://archtheme.com/loginstyle) version.

= One click templates are not working. =

Please make sure the server has read and write permissions for the folder `wp-content/plugins/loginstyle/public/css/`.
